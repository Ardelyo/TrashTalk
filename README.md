## 🌟 **Tentang TrashTalks**

> *"Transformasi digital untuk lingkungan yang berkelanjutan"*

**TrashTalks** adalah aplikasi web progresif (PWA) revolusioner yang menggabungkan kekuatan kecerdasan buatan dengan kepedulian lingkungan. Dengan teknologi vision AI terdepan, aplikasi ini mampu menganalisis sampah secara visual dan memberikan informasi komprehensif tentang material, tingkat daur ulang, dampak lingkungan, serta solusi konkret yang dapat diterapkan.

### 💡 **Visi & Misi**
- **Visi**: Menjadi platform digital terdepan dalam edukasi dan pengelolaan sampah berkelanjutan
- **Misi**: Memberdayakan masyarakat Indonesia dengan teknologi AI untuk menciptakan lingkungan yang lebih bersih dan sehat

## 📥 **Instalasi & Penggunaan**

### **Persyaratan Sistem**
- 🌐 Browser modern (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- 📱 Perangkat dengan kamera (opsional)
- 🌍 Koneksi internet untuk API AI
- 🔑 API Key dari penyedia AI pilihan Anda

### **🚀 Quick Start**

#### **1. Clone Repository**
```bash
git clone https://github.com/username/trashtalks.git
cd trashtalks
```

#### **2. Setup Development Server**
```bash
# Menggunakan Python
python -m http.server 8000

# Menggunakan Node.js (http-server)
npx http-server -p 8000

# Menggunakan PHP
php -S localhost:8000
```

#### **3. Konfigurasi API**
1. 🔧 Buka aplikasi di `http://localhost:8000`
2. ⚙️ Klik ikon **Settings** (⚙️) di pojok kiri bawah
3. 🔑 Masukkan konfigurasi AI Anda:
   - **Provider**: Pilih OpenRouter, OpenAI, atau Google AI
   - **API Key**: Masukkan API key yang valid
   - **Model**: Spesifikasi model AI (contoh: `gpt-4o`, `google/gemma-2-9b-it`)
4. 💾 Klik **Simpan** dan mulai analisis!
